피트스탑 적합성 — 조건부 적합 (선결 7조건 충족 전제)

한 줄 요약: 금융권 AI 에이전트 개발사 모트에이아이는 매출 11.8억, 회계 기준 영업이익 1.2억 흑자(현금 기준 연 3.3억 흑자)를 내는 구조조정과 사업 피벗팅을 통해 펀더멘탈이 양호해지고 있는 기업입니다. 그러나 과거 투자 유치·사업 확장 과정에서 쌓인 부채(오버행 88.8억) + 완전자본잠식(-29.4억) + 잔고 1,436만원의 3중 자본 위기 상태입니다.

핵심 판단: 돈을 벌 수 있는 사업인가? → Yes (현금 흑자, 금융사 레퍼런스 다수). 자금을 넣어도 안전한가? → 조건부 (기존 CB 전환 + 은행 대환 + RCPS 유예가 선행되어야 함). 이 조건들이 충족되지 않으면 5억 투입은 88.8억 오버행 뒤 최후순위 채권이 됩니다. 선결 조건 충족 후 투자 시 기본 시나리오 기대 배수 3.6x (IRR 38%).

2. 피트스탑 판정 — 받을 수 있는가?

피트스탑 투자를 집행하기 전, 반드시 충족해야 하는 선결 조건을 정리합니다. 선결 조건이란 ‘이것이 안 되면 투자 자체가 의미 없는’ 최소 요건입니다. Tier 1(즉시 확인)과 Tier 2(구조 설계 전 확인)로 구분하며, Tier 1이 하나라도 불통과되면 투자를 진행하지 않습니다.

6.3 최종 결론

Q. 피트스탑을 받을 수 있는가?

조건부 YES. Tier 1 선결 3조건(대환 확정, RCPS 유예, TP-핀테크 동의) 충족 시 GO. 하나라도 불통과 시 DROP.

Q. 메자닌 집행 시 원금+이자를 받으려면?

현금 BEP 연 3.3억이 유지되면 원금 5억 + 이자 0.8억 = 5.8억(1.16x) 회수 가능. CB 전환 선행이 회수 안전성의 핵심.

Q. 주식 전환되려면?

2026 매출 25억+ 확인 + CB 25억 전환 완료 + 영업이익 5억+ 시 전환 유리. 최적 시점 T+18개월(2027 Q3) — 상각 종료 전 저평가 구간.

Q. Exit 시 기대 수익률은?

기본 시나리오(35억 매출) 기준 3.6x / IRR 38%. 보수(20억)에서도 1.7x. 최대 리스크는 CB 미전환 + RCPS 상환 청구(원금 전액 손실).

Q. 전체 리스크 대비 수익률은?

확률 가중 기대값 12.0억(2.4x). 다만 이는 Tier 1 충족 전제. 미충족 시 파산 확률 30%로 급등하여 기대값 0.9x(원금 손실).

피트스탑 진행 순서 권고

  1. Tier 1 확인 (1주): 대환 확정 + RCPS 유예 + TP-핀테크 동의
  2. CB 전환 협상 (1~2개월): 24명 전환 의향 파악, 전환 조건 확정
  3. 전환 완료 후 5억 메자닌 집행
  4. 대환 + 운전자금 확보 → 이자비용 절감 → 매출 성장 가속
  5. T+18mo 출자전환 검토 → T+24~36mo Exit
